# Sahid Fitness - Premium Gym Website

A modern, fully responsive multi-page website built for a premium fitness center. This project showcases structured layouts, smooth navigation, and clean design adapted for desktop and mobile devices.

## 🔗 Live Demo
Check out the live deployment here: **[Sahid Fitness Live Demo](https://gym-website-pi-eight.vercel.app/)**

## 🚀 Features
* **Fully Responsive Design:** Tailored layouts for desktops, tablets, and mobile devices.
* **Modern Navigation:** Interactive mobile hamburger menu for seamless user experience.
* **Structured Information:** Dedicated pages for fitness programs, facility details, and contact options.
* **User Engagement:** Structured dynamic inquiry form for potential gym members.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Grid/Flexbox architecture), Vanilla JavaScript
* **Backend Utilities (Local):** PHP Form Handlers (Inquiry management)

## 📸 Project Architecture
* `assets/` - Image optimizations and graphical banners
* `css/` - Split stylesheets (`base.css`, `components.css`, `pages.css`) for high maintainability
* `pages/` - Content pages (`about.html`, `contactus.html`, `fitnesscentre.html`)
* `php/` - Server-side scripts for local validation
